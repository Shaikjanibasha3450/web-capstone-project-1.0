# Web Capstone Project 1.0

A simple HTML portfolio showcasing personal and professional information in a clean, accessible format.

## Check the website 
  [`Go to Official Website`](https://shaikjanibasha3450.github.io/web-capstone-project-1.0/)

## Features

- **Personal Info**: Displays name and profile image
- **Career Summary**: Overview of professional background and expertise
- **Education Section**: Academic qualifications and institutions
- **Project Gallery**: Highlights of notable projects
- **Skills Section**: Technical skills and competencies
- **Responsive Layout**: Simple, browser-compatible design

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Shaikjanibasha3450/web-capstone-project-1.0.git
   ```
2. Navigate to the project directory:
   ```bash
   cd web-capstone-project-1.0
   ```

## Usage

To view the portfolio, simply open the `index.html` file in any web browser:

- **Option 1**: Double-click on `index.html` to open it in your default browser
- **Option 2**: Right-click on `index.html` and select "Open with" your preferred browser
- **Option 3**: Drag and drop `index.html` into an open browser window
